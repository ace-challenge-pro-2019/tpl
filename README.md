## 方針や目的
- チャットツールは報告作業に用いられるが、タスク整理（Todo担当者変更）やファイル修正は別ツールでしか行えない
  - 結局、一連の作業工程を実施するのに、複数ツールを使うことになる。
  - 煩雑
それを、チャットツール上で完結したい＆他ツールとの互換性を高めたい

## 使用形態
PCブラウザ（第一フェーズ）
